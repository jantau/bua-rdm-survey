0.1 Reproduce the report

This report was created with Bookdown in GitBook style. A good introduction to Bookdown offers bookdown: Authoring Books and Technical Documents with R Markdown.

To reproduce the report, download the latest version from Zenodo or fork repository from GitHub.

  1. Open bookdown.Rproj

  2. Render report with bookdown::render_book(new_session = TRUE)
